# Screenplay Templates

This directory contains reusable templates for screenplay documents and tracking.

## Available Templates

### Screenplay Templates
- `screenplay-template.fountain` - Basic screenplay structure
- `title-page.fountain` - Title page metadata

### Tracking Templates
- `character-registry.md` - Character tracking
- `timeline-registry.md` - Timeline tracking
- `location-registry.md` - Location tracking
- `prop-registry.md` - Prop tracking

### Analysis Templates
- `beat-sheet.md` - Story beat analysis
- `scene-breakdown.md` - Scene-by-scene breakdown
- `character-arc.md` - Character arc tracking

## Using Templates

### Copy and Customize
1. Copy template to your working area
2. Replace placeholder text
3. Fill in project-specific details

### Template Placeholders
Templates use these placeholder conventions:
- `[TITLE]` - Replace with screenplay title
- `[CHARACTER]` - Replace with character name
- `[LOCATION]` - Replace with location name
- `[DATE]` - Replace with current date
- `[DESCRIPTION]` - Replace with description

## Creating New Templates

When creating new templates:
1. Use clear placeholder syntax
2. Include usage instructions
3. Provide example content
4. Add to this README
